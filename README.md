# 极简密码

## 使用了RxJava、RxAndroid、RxBinding、ButterKnife、SqlCipher、AppIntro、FoldingCell、ProgressHint等开源库，设计遵循Material Design，简化了操作来契合极简的风格。首次启动应用会显示引导页，引导页结束后进入设置密码界面，每次进入应用需要输入正确密码，用户的每项数据都存放在强加密数据库中来保障安全
